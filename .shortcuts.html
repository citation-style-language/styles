Validation

XML validator Validator.nu.

  1. Enter the link to the CSL schema file in the “Schemas” text field.
    For CSL 1.0 (Zotero 2.1 and up), this is
    https://github.com/citation-style-language/schema/raw/master/csl.rnc  (click here to view)
    or http://bitbucket.org/bdarcus/csl-schema/raw/tip/csl.rnc  (click here to view)
  2. Select either “XML: don’t load external entities” or “XML: load external entities” in the “Parser” drop down menu.
  3. Click the “Validate” button. If the style is correct, you will get the message “The document validates according to the specified schema(s).”
